# 规格（Spec）总览

本目录是 topic-level specification catalog：每个 spec 对应一个稳定主题、能力、接口或约束的长期契约，而不是单次任务的进度卡片。

每个 spec 目录默认包含：

- `SPEC.md`：当前有效规范 / 主题契约
- `IMPLEMENTATION.md`：实现覆盖、当前状态、剩余缺口与 rollout 事实
- `HISTORY.md`：关键演进原因与决策级历史
